# Quick Start


## Installation

### From binary
See [Downloads Page](/downloads/index.md)

### From source

#### Dependencies
- [DMD](https://dlang.org/download.html)
- [Xmake](https://xmake.io/)

#### Build
```sh
xmake build
```

#### Install
```sh
xmake install
```
